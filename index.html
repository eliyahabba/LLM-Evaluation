Abstract

DOVE is a large-scale dataset for studying how Language Models (LLMs) respond to different ways of asking the same question. With over 300M model predictions, DOVE helps researchers understand and improve LLM evaluation. The dataset covers 78 popular benchmarks and includes comprehensive prompt variations across 5 dimensions.

Key Features

  • Comprehensive prompt variations across 5 dimensions
  • Coverage of 78 popular benchmarks (MMLU, ARC, HellaSwag, etc.)
  • Multiple model families (Llama, OLMo, Mistral)
  • Zero-shot and few-shot evaluations
  • Full token probability logging
